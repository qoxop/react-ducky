React-Ducky 是一个用来优化 React 应用的状态管理的工具库。它可以帮你便捷地管理应用的数据模型(不管是同步的还是异步的)。同时也支持将任何状态数据与路由堆栈进行绑定，从而减少你对缓存管理的烦恼 。
